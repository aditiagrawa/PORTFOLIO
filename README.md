## About

This is my personal developer portfolio — a single-page site styled like a commit log, covering my work across AI/ML engineering, full-stack development, and data science. It's built from scratch with plain HTML, CSS, and JavaScript, no frameworks or build step required.

The design leans into a "career as version history" theme: experience entries read like commits with diff-style metrics, projects are laid out like repositories, and skills are organized like a manifest file.

## Features

- **Voice narration** — an in-browser narrator (using the Web Speech API) walks visitors through the portfolio section by section, with auto-scroll, live captions, and a clickable progress bar.
- **Fully responsive** — adapts from desktop down to mobile.
- **No dependencies** — single HTML file, no build tools, no npm install.
- **Accessible** — visible keyboard focus states, respects `prefers-reduced-motion`.

## Tech Stack

`HTML5` · `CSS3` · `JavaScript (ES6)` · `Web Speech API` · `IntersectionObserver`

## Running Locally

No build step needed — just open the file directly, or serve it locally:

```bash
git clone https://github.com/aditiagrawa/portfolio.git
cd portfolio
# open index.html directly in a browser, or:
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploying

This repo is set up to deploy straight to GitHub Pages:

1. Push `index.html` to the `main` branch.
2. Go to **Settings → Pages**.
3. Under **Source**, select the `main` branch and `/ (root)`.
4. Your site will be live at `https://aditiagrawa.github.io/<repo-name>/` within a couple of minutes.
